# Firefox ::-moz-focus-inner

对于点击某些元素后出现的边框，去除的办法是:

button:focus {
    outline: none;
}
1
2
3

但是在 Firefox 下，对于类型为button的元素，即使设置outline，也还是会出现虚线。这个时候就需要用 Firefox 的私有伪元素::-moz-focus-inner：

button::-moz-focus-inner {
    border: 0;
}
